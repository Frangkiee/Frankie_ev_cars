# DESIGN.md

## 产品定位
2026年纯电新车速递 - 纯电动车型信息展示站，面向汽车爱好者与潜在购车用户。

## 气质与意象
深夜城市展厅：想象午夜时分走进一间极简展厅，聚光灯打在每辆车上，霓虹绿色指示灯在深色地面上反射光泽。安静、高级、信息密度高但不杂乱。

## 配色方案
- 背景色：近黑深灰 `#0a0a0a` / `#111111` — 展厅地面质感
- 卡片背景：微亮深灰 `#1a1a2e` / `#16161a` — 展台基座
- 强调色（新能源绿）：`#00e5a0` / `#10b981` — 充电桩指示灯色
- 价格高亮：琥珀金 `#f59e0b` — 高端质感
- 文字层级：白色 `#f0f0f0` > 浅灰 `#a0a0a0` > 暗灰 `#666`
- 卡片边框：微光描边 `rgba(255,255,255,0.06)`

## 字体排版
- 标题：粗体，大字号，紧凑字间距
- 续航里程数字：超大加粗，强调数据感
- 标签：小号大写字母风格，半透明背景
- 整体节奏：留白充足，信息分层清晰

## 动效与交互
- 卡片 hover：微微上浮 + 边框发光（绿色辉光）
- Tab 切换：底部滑块过渡
- 过渡曲线：`cubic-bezier(0.4, 0, 0.2, 1)`

## 设计禁忌
- 不要使用科技蓝+紫色渐变（太通用）
- 不要白色/浅色背景
- 不要过度装饰、渐变堆叠
- 不要密集排列，保持呼吸感
