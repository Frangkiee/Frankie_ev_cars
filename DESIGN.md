# DESIGN.md

## 产品定位
2026年6月纯电新车数据表 - 面向汽车爱好者与潜在购车用户的纯电车型数据展示站。

## 气质与意象
数据仪表盘：深色背景上的高密度数据表格，类似金融终端或赛车遥测界面。信息密集但层次分明，用颜色标签快速区分关键属性。

## 配色方案
- 背景色：近黑深灰 `#0a0a0a` / `#111115` — 深色终端感
- 表头背景：`#1a1a2e` — 深蓝黑，突出表头层级
- 奇偶行：`#111115` / `#0d0d11` — 微妙交替，不刺眼
- 强调色（新能源绿）：`#00e5a0` — 用于续航数据、统计数字
- 价格高亮：琥珀金 `#f59e0b` — 加粗醒目
- 电池标签：LFP 翠绿 / NCM 天蓝
- 驱动标签：FWD 天蓝 / RWD 橙色 / AWD 玫红
- 文字层级：白色 `#f0f0f0` > 浅灰 `#a0a0a0` > 暗灰 `#666`

## 字体排版
- 数字列：等宽数字 `tabular-nums`，右对齐
- 车名：加粗白色，视觉焦点
- 标签：小号 + 半透明背景 + 彩色文字
- 整体节奏：行间距紧凑，数据密度高

## 动效与交互
- 行 hover：绿色微光高亮 `bg-[#00e5a0]/[0.06]`
- 过渡曲线：`cubic-bezier(0.4, 0, 0.2, 1)`
- 表格支持横向滚动（移动端）

## 设计禁忌
- 不要使用科技蓝+紫色渐变
- 不要白色/浅色背景（页面整体）
- 不要过度装饰、渐变堆叠
- 不要卡片式布局（本阶段为表格风格）
